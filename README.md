# موقع خدمات تقنية المعلومات

**النطاق:** wr-454.com

موقع ويب ثابت بسيط بلغة HTML وCSS وJavaScript مصمم لعرض خدمات تقنية المعلومات.

## المحتويات

- `index.html` – الصفحة الرئيسية.
- `styles.css` – أنماط التصميم.
- `script.js` – تفاعل القائمة ونموذج الاتصال.

## التشغيل

افتح `index.html` في المتصفح لعرض الموقع.
